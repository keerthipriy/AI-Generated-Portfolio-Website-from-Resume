# AI-Generated-Portfolio-Website-from-Resume
An AI-powered project that converts a resume into a professional, responsive portfolio website. It extracts key details like skills, projects, education, and experience, and organizes them into a clean web layout, reducing manual effort and showcasing AI-driven automation in web development.

The extracted data is then structured into predefined portfolio sections like About Me, Skills, Projects, Experience, and Education. A modern and responsive web interface is dynamically generated using web technologies, ensuring compatibility across devices. This eliminates the need for manual website creation and helps job seekers quickly establish an online presence.

The project highlights practical applications of AI in automation, text processing, and full-stack development, making it especially useful for students and professionals looking to showcase their profiles efficiently.

# Key Highlights:
Resume parsing using AI/NLP
Automatic content structuring (About, Skills, Projects, Experience)
Responsive and modern UI design
Reduces manual effort in portfolio creation
